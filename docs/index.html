MONA

MONA is a React Native app built with Expo that allows users to upload items they want to give away for free. The app fosters a community of sharing, allowing people to request items they need and receive notifications when items are available.

MONA is a brand registered in South Korea under MONA Co., Ltd. The company website is monaofficial.co.

The original framework, design of the app, and code was developed by Josie Daw as a basic MVP for MONA.

Features

  • Upload Items: Users can upload items they want to give away, including descriptions and photos.
  • Request Items: Users can request available items that are being given away by other users.
  • Notifications: Users receive notifications when an item they requested is available.
  • Browse Items: Users can browse all available items that people are giving
  • Profile Management: Users can manage their profiles, including their listed and requested items.

Installation

To get started with the MONA app, follow these steps:

  1. Clone the repository:
git clone https://github.com/JosDaw/mona.git
cd mona
  1. Install dependencies:

Make sure you have Node.js and Expo CLI installed on your machine.

Install the required dependencies by running:


npm install
  1. Start the Expo server:

Start the Expo development server using the command:

npx expo start

This will launch the Expo Metro. You can then run the app on your iOS or Android device by scanning the QR code or use an emulator.

Usage

Sign Up / Log In

  • New users can sign up with their email and password.
  • Existing users can log in to their accounts.

Upload Items

  • To give away an item, go to the "Give" section, fill in the item details (title, description, category, etc.), and upload a photo of the item.

Browse and Request Items

  • Browse items that are being given away in your community.
  • If you find an item you want, simply click the "Request" button.

Get Notifications

  • You'll receive a notification when an item you requested becomes available or when a new item matching your request is uploaded.

Manage Your Profile & Requests

  • View profile information and upgrade to MONA Plus.
  • View items you've uploaded and requested.

Technologies Used

  • React Native: Cross-platform mobile app framework for building the app.
  • Expo: A framework and platform for universal React applications.
  • Firebase: For authentication, storage, and real-time database needs.
  • Push Notifications: For notifying users about their requested items.
  • EmailJS & MailJet: For sending email notifications to users.
  • Sentry: For error monitoring and tracking.
